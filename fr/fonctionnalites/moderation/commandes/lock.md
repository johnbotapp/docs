# Lock

{% hint style="info" %}
Vous devez avoir la permission de gérer les salons pour utiliser cette commande.
{% endhint %}

La commande Lock permet lorsqu'elle est exécutée correctement de retirer la permission d'envoyer des messages au rôle **@everyone** (voir image #1).

Syntaxe de la commande : /lock \[salon]
