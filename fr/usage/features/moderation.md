### :rocket: Introduction

Le système de modération par commandes de John-Bot vous donne les outils pour modérer efficacement votre serveur Discord. Grâce à des commandes puissantes, vous pouvez simplifier et accélérer la modération pour maintenir un environnement sécurisé, le tout avec des fonctionnalités avancées via John-Bot inaccessibles aux utilisateurs classiques.

### :clipboard: Liste des commandes
| Commande | Description | Exemple |
| -------- | ----------- | ------- |
| /ban | Banni un utilisateur du serveur. | ![Commande /ban](../../.gitbook/assets/moderation_command_ban.png) |
| /clear | Supprime un nombre de messages dans un salon. | ![Commande /clear](../../.gitbook/assets/moderation_command_clear.png) |
| /kick | Expulse un membre du serveur. | ![Commande /kick](../../.gitbook/assets/moderation_command_kick.png) |
| /lock | Verrouille un salon du serveur. | ![Commande /lock](../../.gitbook/assets/moderation_command_lock.png) |
| /mute | Réduit un membre au silence pendant un certain temps. | ![Commande /mute](../../.gitbook/assets/moderation_command_mute.png) |
| /nickname | Change le surnom d'un membre du serveur. | ![Commande /nickname](../../.gitbook/assets/moderation_command_nickname.png) |
| /slowmode disable | Désactive le mode lent d'un salon. | ![Commande /slowmode disable](../../.gitbook/assets/moderation_command_slowmode%20disable.png) |
| /slowmode set | Défini le mode lent d'un salon. | ![Commande /slowmode set](../../.gitbook/assets/moderation_command_slowmode%20set.png) |
| /unban | Révoque le bannissement d'un utilisateur. | ![Commande /unban](../../.gitbook/assets/moderation_command_unban.png) |
| /unlock | Déverrouille un salon du serveur. | ![Commande /unlock](../../.gitbook/assets/moderation_command_unlock.png) |
| /unmute | Rend la parole à un membre réduit au silence. | ![Commande /unmute](../../.gitbook/assets/moderation_command_unmute.png) |
| /warn | Averti un membre du serveur. | ![Commande /warn](../../.gitbook/assets/moderation_command_warn.png) |
| /warnings clear | Supprime les avertissements d'un membre ou de tous les membres du serveur. | ![Commande /warnings clear](../../.gitbook/assets/moderation_command_warnings%20clear.png) |
| /warnings list | Affiche la liste des avertissements d'un membre. | ![Commande /warnings list](../../.gitbook/assets/moderation_command_warnings%20list.png) |
| /warnings remove | Supprime l'avertissement d'un membre. | ![Commande /warnings remove](../../.gitbook/assets/moderation_command_warnings%20remove.png) |
