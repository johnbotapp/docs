## Calculer l'expérience nécessaire pour atteindre le niveau supérieur

La formule suivante est utilisée : `5 * (lvl ^ 2) + (50 * lvl) + 100 - xp`

Pour :
* `lvl` : niveau actuel
* `xp` : experience actuelle
