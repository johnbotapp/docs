# Moderation

**Video tutorial related to this page:** [Effectively Moderate Your Server with John-Bot - Tutorial #5](https://jnbt.xyz/tutorials/moderation)

### :rocket: Introduction

John-Bot's command-based moderation system gives you the tools to efficiently moderate your Discord server. With powerful commands, you can simplify and speed up moderation to maintain a safe environment â€” all with advanced features through John-Bot that are unavailable to regular users.

### :clipboard: Command List
| Command | Description | Example |
| ------- | ----------- | ------- |
| /ban | Bans a user from the server. | ![/ban command](../../.gitbook/assets/moderation_command_ban.png) |
| /clear | Deletes a number of messages in a channel. | ![/clear command](../../.gitbook/assets/moderation_command_clear.png) |
| /kick | Kicks a member from the server. | ![/kick command](../../.gitbook/assets/moderation_command_kick.png) |
| /lock | Locks a server channel. | ![/lock command](../../.gitbook/assets/moderation_command_lock.png) |
| /mute | Mutes a member for a certain duration. | ![/mute command](../../.gitbook/assets/moderation_command_mute.png) |
| /nickname | Changes a server member's nickname. | ![/nickname command](../../.gitbook/assets/moderation_command_nickname.png) |
| /slowmode disable | Disables slowmode in a channel. | ![/slowmode disable command](../../.gitbook/assets/moderation_command_slowmode%20disable.png) |
| /slowmode set | Sets slowmode in a channel. | ![/slowmode set command](../../.gitbook/assets/moderation_command_slowmode%20set.png) |
| /unban | Unbans a user. | ![/unban command](../../.gitbook/assets/moderation_command_unban.png) |
| /unlock | Unlocks a server channel. | ![/unlock command](../../.gitbook/assets/moderation_command_unlock.png) |
| /unmute | Unmutes a muted member. | ![/unmute command](../../.gitbook/assets/moderation_command_unmute.png) |
| /warn | Warns a server member. | ![/warn command](../../.gitbook/assets/moderation_command_warn.png) |
| /warnings clear | Clears warnings for a member or all server members. | ![/warnings clear command](../../.gitbook/assets/moderation_command_warnings%20clear.png) |
| /warnings list | Shows a member's warning list. | ![/warnings list command](../../.gitbook/assets/moderation_command_warnings%20list.png) |
| /warnings remove | Removes a warning from a member. | ![/warnings remove command](../../.gitbook/assets/moderation_command_warnings%20remove.png) |
